# Беспроводной роутер Bitcord RTR-3

Промышленный двухсимочный беспроводной роутер LTE/4G, WiFi. Предназначен для применения в качестве модуля связи, а также устройства приёма и передачи телеметрических данных.

## Основные характеристики

- 4G, скорость до 100 Мб/с
- Поддержка Wi-Fi 802.11b/g/n
- 5xEthernet-портов 10/100, WAN/LAN
- 2xUSB слота
- 1xMicroSD-слот
- Интерфейсы RS-232, RS-485, GPIO
- GRE-IPsec-OpenVPN- туннели

## Аппаратная часть

| Параметр                  | Значение           |
| ------------------------- | ------------------ |
| **`Процессор`**           | MIPS 24KEc 580 Mhz |
| **`Динамическое ОЗУ`**    | 128 MБ             |
| **`Сотовый модуль`**      | 1 х LTE Cat 4      |
| **`Количество SIM-карт`** | 2 шт.              |
| **`Объем flash-памяти`**  | 32 Мб              |
