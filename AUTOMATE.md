# Автопереключение SIM-карт

В случае перебоев со связью, двухсимочный LTE/4G модем автоматически переключается на резервную сим-карту.

## Настройка пользователем

### Список параметров

| Название                 | По умолчанию | Аннотация                                                          |
| ------------------------ | ------------ | ------------------------------------------------------------------ |
| **`Уровень сигнала`**    | `30%`        | Минимальный уровень сигнала до базовой станции сотового опрератора |
| **`Минимальный баланс`** | `50р`        | Минимальная сумма денежных средств на балансе SIM-карты            |
